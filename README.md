Cree un juego RPG simple utilizando POO. 
Tomé inspiración de la novela ligera "Hai to Gensou no Grimgar"

Empecé a escribir código en este orden:
-dominio = Lo que se define para el juego
-motor = Las interacciones dentro del juego
-user_interface (ui) = La visualización
-main = El punto de entrada
